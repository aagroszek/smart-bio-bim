Where is this hosted?

Data flow, storage & cost breakdown
🌐
The dashboard — runs entirely in your browser (no server)

This is a single self-contained HTML file. It runs locally in any browser with no backend, no database, and no login. Open it from your desktop, or host it for free on Netlify, Vercel, GitHub Pages, or Cloudflare Pages. BIM instance data lives in JavaScript memory — it resets on page refresh unless you export it (CSV/JSON buttons available).

🤖
Myra AI Agent — Anthropic API (Claude Sonnet)

Chat messages are sent to api.anthropic.com/v1/messages. No conversation is stored between sessions — your browser holds the chat history only while the tab is open. API calls use your Anthropic account's pay-as-you-go quota. At ~$3/M input + ~$15/M output tokens, a full working day of queries costs under $0.05.

📦
BIM tracking data — currently in-memory only

Instance counts, model names, and carbon totals are stored in JS variables. For persistent cross-session tracking and live sync across multiple users (e.g. live data visible from Revit/Rhino), connect to a free-tier database: Supabase (500MB free, REST API), Airtable (free tier), or a simple JSON REST API on Render. That would also enable Revit dynamo scripts or Rhino Grasshopper components to push placement events and pull live impact totals.

📐
Exporting to Revit / Rhino

Use the IFC export to load directly into Revit, ArchiCAD, or any IFC-compliant BIM tool — all EPD properties are embedded as IFC Psets. Use the OBJ export for Rhino, 3ds Max, Blender, or SketchUp. The JSON export contains the full smart data payload and can be consumed by Revit Dynamo scripts or Grasshopper via the HttpClient component to pull live impact data back into this dashboard.

✓ Free to run locally — zero infrastructure needed

Free hosting stack: Netlify / GitHub Pages / Cloudflare Pages (all free for static HTML).
AI cost: Pay-as-you-go Anthropic API — no subscription. A full day of agent queries ≈ $0.02–0.10.
Persistent tracking (optional upgrade): Supabase free tier is sufficient for hundreds of BIM projects.
